# Steam DWProton

Updated: 2026-05-03_01-42-12

DWProton is installed automatically into Steam compatibility tools.

Managed version:

```text
dwproton-10.0-26
```

Possible targets:

```bash
~/.steam/root/compatibilitytools.d/dwproton-10.0-26
~/.local/share/Steam/compatibilitytools.d/dwproton-10.0-26
```

Manual install:

```bash
bash scripts/install-steam-dwproton.sh
```

Restore helper automatically runs:

```bash
bash scripts/install-from-current-system-snapshot.sh
```

After install, restart Steam and select:

```text
DWProton 10.0-26
```
